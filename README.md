# 打地鼠游戏

一个基于HTML、CSS和JavaScript的经典打地鼠游戏。

## 游戏说明

1. 点击"开始游戏"按钮开始游戏
2. 游戏时长为30秒
3. 地鼠会随机从洞中冒出，点击地鼠得分
4. 随着时间的推移，地鼠出现的速度会变快
5. 游戏结束后会显示最终得分

## 如何运行

直接在浏览器中打开`index.html`文件即可开始游戏。

## 游戏特点

- 响应式设计，适配不同屏幕尺寸
- 点击效果动画
- 难度递增
- 简洁直观的界面

## 技术栈

- HTML5
- CSS3
- JavaScript (原生)
